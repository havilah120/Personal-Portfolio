New Horizons Website Redesign

UI/UX Case Study

Project Overview

New Horizons is a digital platform designed to improve user engagement and accessibility. The goal of this project was to redesign the website experience to be modern, intuitive, and visually engaging while maintaining clarity and usability.

The Problem

The previous design lacked structure, had poor visual hierarchy, and made it difficult for users to quickly find information. Navigation was cluttered, and the interface did not reflect a modern brand identity.

The Solution

I redesigned the interface with a clean layout, improved spacing, and a modern color palette. The new design focuses on clarity, accessibility, and smooth user flow.
